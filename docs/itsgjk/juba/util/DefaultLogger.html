Class DefaultLogger

  • java.lang.Object
    • org.slf4j.helpers.MarkerIgnoringBase

  • All Implemented Interfaces:
    java.io.Serializable, org.slf4j.Logger

    public class DefaultLogger
extends org.slf4j.helpers.MarkerIgnoringBase
    Code inspiration from SLF4J's SimpleLogger, made to be simple and have just enough features for JUBA. It is recommended to override JUBA's default logger with a different one, such as Log4J2.
